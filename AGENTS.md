# Aegis Toolchain

This is the **Aegis Toolchain** repository. Aegis governs its own development.

## For AI Assistants

Before any change to aegis-toolchain source:

1. Read `Aegis/skills/aegis-boot/SKILL.md`
2. Classify the change (L1/L2/L3)
3. Use `aegis start "<title>" -l <L1|L2|L3>` to register

## Quick Commands

```bash
aegis status     # View current state
aegis check      # Run boundary check
```
