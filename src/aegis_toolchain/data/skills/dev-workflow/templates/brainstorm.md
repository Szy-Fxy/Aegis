# 01-brainstorm 模板

> L3-1 阶段：识别问题、提出方案、不写实现细节。

```markdown
# 头脑风暴

## 问题
[用 1-3 句话描述当前痛点/需求。Bug 需描述复现路径。]

## 方案
1. **方案 A — {简述}**
   - 特点：{一句话核心思路}
   - 风险：{最大风险点}

2. **方案 B — {简述}**
   - 特点：{一句话核心思路}
   - 风险：{最大风险点}

## 推荐
[AI 建议的方案 + 简要理由，格式：推荐方案 X，因为...]
```

**规则**：
- 不写实现细节、不写 API 签名
- 必须点出根本原因，不只描述表面现象
- 至少给 2 个方案对比
- Bug 需包含复现路径
